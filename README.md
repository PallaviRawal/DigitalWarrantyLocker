# 💼 Digital Warranty Locker

**Digital Warranty Locker** is a smart web-based application designed to help users securely store and manage their product warranties, bills, and receipts in one place. It provides features like OCR-based bill scanning, warranty renewal reminders, and email notifications — all within a simple, user-friendly dashboard.

---

##  Features

- 📦 **Digital Storage:** Save and manage product warranties and bills digitally.  
- 🧠 **OCR Bill Scanning:** Automatically extract product details from uploaded bills.  
- ⏰ **Renewal Reminders:** Get notified before a product’s warranty expires.  
- 📧 **Email Notifications:** Users receive email alerts on successful registration and when a warranty is about to expire.  
- 📊 **Dashboard Analytics:** View warranty statistics and insights in an organized format.  

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (XAMPP Server)  
- **Libraries:** Tesseract OCR (for bill scanning)  
- **Server:** Apache (via XAMPP)  
- **Email Service:** PHP Mailer / SMTP

---
